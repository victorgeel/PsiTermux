# PsiTermux

Another Method for Termux with Psiphon pro

# Installation 

>$ apt update
>
>$ apt upgrade
>
>$ pkg install python
>
>$ pkg install python2
>
>$ pip2 install requests
>
>$ pip2 install mechanize
>
>$ pkg install git
>
>$ gir clone
>

# Run

>$ cd freenet
>
>$ chmod +x Main.py
>
>$ python2 Main.py

Enjoy
