# PsiTermux

Another Method for Termux with Psiphon pro

# Installation 

>$ apt update
>
>$ apt upgrade
>
>$ pkg install python
>
>$ pkg install python2
>
>$ pip2 install requests
>
>$ pip2 install mechanize
>
>$ pkg install git
>
>$ git clone https://github.com/victorgeel/PsiTermux.git


# Run

>$ cd PsiTermux
>
>$ chmod +x *
>
>$ python2 main.py
